Sistema de Gestión — Bodega Diana

Sistema de inventario y ventas en Python con persistencia en JSON.

Requisitos


Python 3.10 o superior
Sin dependencias externas (solo librería estándar)


Estructura

Bodega/
├── main.py
├── data/
│   └── inventario.json
└── Modulos/
    ├── config.py
    ├── archivo.py
    ├── operacion.py
    ├── Ventas.py
    └── reportes.py

Ejecución

bashpython main.py

Al iniciar responde SI para mostrar el menú principal.

Funcionalidades


Buscar producto y registrar venta desde el resultado
Registrar venta directa con validación de stock
Ver inventario completo
Gestionar mercadería: agregar, actualizar y eliminar productos
Generar reportes por fecha: cantidad de ventas, producto más vendido e ingresos/ganancias
